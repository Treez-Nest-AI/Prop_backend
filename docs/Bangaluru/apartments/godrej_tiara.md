# Godrej Tiara


## Description
Godrej Tiara is a premium residential apartment community located in Yeshwanthpur, West Bangalore. Developed by Godrej Properties, the project offers spacious 3 BHK and 4 BHK homes designed with contemporary architecture and premium specifications.
